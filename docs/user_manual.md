# KidRead 使用说明

## 1. 创作页
1. 进入“创作”页。
2. 输入文字、语音或点击选项发起创作。
3. 系统会生成 lead / story / guide 三段式回复。
4. 点击“归档”可将当前创作保存为新故事。

## 2. 书籍页
1. 从书架打开一本书。
2. 点击“听”可播放朗读。
3. 点击“聊”进入书内对话。

## 3. 书内对话页
1. 输入问题或续写要求。
2. 系统会自动读取该书的 StorySpec / StoryState / StorySummary。
3. 若内容超出年龄分级或安全规则，后端会在最终结果阶段自动改写。
4. 点击归档可把本轮草稿合并进正式正文。

## 4. 历史会话
- 支持会话标题自动生成
- 支持置顶、重命名、删除
- 会保存草稿内容和上下文快照
